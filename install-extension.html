install-extension

Quickinstall.red Vscode Extension

Purpose

install vscode extension from name or url. Log the extension for reinstalling from log file.
For example: 


install-extension https://marketplace.visualstudio.com/items?itemName=2gua.rainbow-brackets

Usage

Requirements:

load library:

do https://quickinstall.red/install-extension

install extension from name(s)

example:

do https://quickinstall.red/install-extension
install-extension "2gua.rainbow-brackets" !!! any name starting with number must be double-quoted

or:

do https://quickinstall.red/install-extension
install-extension [
    "2gua.rainbow-brackets" ; !!! any name starting with number must be double-quoted
    inu1255.easy-snippet
]

install extension from url(s)

example:

do https://quickinstall.red/install-extension
install-extension https://marketplace.visualstudio.com/items?itemName=inu1255.easy-snippet

or:

do https://quickinstall.red/install-extension
install-extension [
    https://marketplace.visualstudio.com/items?itemName=2gua.rainbow-brackets
    https://marketplace.visualstudio.com/items?itemName=inu1255.easy-snippet
]

install extensions from log file or urls list file (COMING SOON...)

see log file location next section

to install from log file in current directory, type this command:

install-extension from-log

to install from log file in any directory or url, use /log-file for example:

install-extension/log-file c:\download\vscode-extensions.log

Log file

vscode-extensions.log

log file is generated under current directory.
